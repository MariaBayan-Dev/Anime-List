# 🎬 Anime List

A web project that displays popular anime using an external API.

## 🚀 Features
- Displays a list of popular anime in a clean gallery layout  
- Simple and easy-to-use interface  

## 🛠️ Technologies
- HTML  
- CSS  
- JavaScript  
![API](https://img.shields.io/badge/API-grey)

## 📚 What I learned
- Working with APIs  
- DOM manipulation  
- Building layouts with HTML and CSS  

## 💡 Future improvements
- Add search functionality  
- Add favorites system  
- Improve UI and visual design
